# DiveDrop OBS Zoom Graphics

תיקיה זו מוכנה לשידור Zoom דרך OBS בהצגת ה־MVP למדריכי צלילה, נהגי שאטלים ומשתמשי צלילה חופשית.

## שימוש מהיר

1. פתח OBS.
2. הוסף Source חדש מסוג `Browser`.
3. סמן `Local file`.
4. בחר אחד מקבצי ה־HTML:
   - `scene-opening.html` לפתיחת ההדגמה.
   - `scene-instructors.html` להצגת מסלול מדריכים.
   - `scene-shuttle-drivers.html` להצגת מסלול נהגי שאטלים.
   - `scene-free-diving.html` להצגת צלילה חופשית והשכרת ציוד.
   - `lower-third.html` לכותרת תחתונה שקופה מעל שיתוף המסך.
5. הגדר גודל `1920 x 1080`.

## הערות הפקה

- כל הגרפיקה מגיעה מהחבילה החדשה בתיקיית `assets`.
- אין תלות באינטרנט.
- מומלץ לשים את סצנת ה־HTML מתחת ל־Camera ומעל Screen Share לפי הצורך.
- ב־lower third הרקע שקוף, לכן הוא מתאים כשכבה מעל הדפדפן.
