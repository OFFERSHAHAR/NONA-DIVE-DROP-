01

קודם בוחרים מסלול

כיום יש מתקין מקורי ל-Claude Code, ולכן Node.js אינו חובה לרוב המשתמשים. מסלול npm עדיין זמין למי שמעדיף אותו.

מומלץ

מתקין מקורי ל-CMD

הדרך הפשוטה והעדכנית ב-Windows, עם עדכונים אוטומטיים וללא תלות ב-Node.js.

  • לא דורש Node.js
  • פקודת CMD אחת
  • מיועד ל-Windows

התקנה דרך npm ב-CMD

מתאים לסביבה שכבר מבוססת Node.js או לניהול חבילות אחיד.

  • דורש Node.js 18 ומעלה
  • דורש npm
  • כל הפקודות מורצות ב-CMD
i
ההמלצה המעשית

התחילו במתקין המקורי. עברו לסעיף Node.js רק אם בחרתם במסלול npm או אם הפרויקט שלכם דורש Node.

02

Node.js — רק למסלול npm

Node.js מריץ JavaScript מחוץ לדפדפן. npm, מנהל החבילות שמגיע איתו, מתקין את Claude Code כחבילה גלובלית.

מה מתקינים?

בחרו גרסת LTS

הורידו LTS מאתר Node.js הרשמי. חבילת npm של Claude Code דורשת Node.js 18 ומעלה.

Node.js הרשמי ↗
1

התקנה

הפעילו את המתקין והשאירו את ברירות המחדל, כולל הוספה ל-PATH.

2

פתחו חלון CMD חדש

כך CMD יטען את נתיב ההתקנה החדש.

3

אימות ב-CMD

שתי הפקודות צריכות להחזיר מספר גרסה.

Windows CMD
C:\> node --version
C:\> npm --version
03

מתקינים דרך CMD

פתחו Command Prompt מתפריט Start והריצו בו את הפקודה המתאימה.

Windows CMD — מתקין מקורי
C:\> curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
Windows CMD — דורש Node.js 18+
C:\> npm install -g @anthropic-ai/claude-code
!
אם מתקבלת שגיאת הרשאות

פתחו CMD כמנהל מערכת, או השתמשו במתקין המקורי שמופיע בלשונית הראשונה.

04

הרצה ראשונה

Claude Code עובד מתוך תיקיית הפרויקט. בהפעלה הראשונה ייפתח תהליך התחברות לחשבון Anthropic.

1

עברו לתיקיית הפרויקט

החליפו את הנתיב בדוגמה בנתיב שלכם.

Windows CMD
C:\> cd C:\Projects\my-app
2

הפעילו מתוך CMD

פקודה אחת פותחת את הממשק האינטראקטיבי.

Windows CMD
C:\Projects\my-app> claude
3

משימה ראשונה

התחילו בבקשת הסבר לפני שינוי קוד.

Claude Code
> Explain this project structure and tell me how to run it
טיפ עבודה

להבין, לתכנן, לבצע

בקשו סקירה, אחר כך תוכנית קצרה, ורק אז שינוי. עברו על ההרשאות וה-diff לפני אישור.

05

בדיקה, אבחון ועדכון

שלוש פקודות שימושיות שחוסכות ניחושים.

פקודות תחזוקה ב-CMD
REM בדיקת גרסה
C:\> claude --version

REM בדיקת בריאות ההתקנה
C:\> claude doctor

REM עדכון התקנת npm
C:\> npm install -g @anthropic-ai/claude-code@latest
✓
הצלחה נראית כך

claude --version מחזיר מספר גרסה, ו-claude פותח ממשק אינטראקטיבי מתוך תיקיית הפרויקט.

06

פתרון תקלות

רוב הבעיות קשורות לטרמינל ישן, PATH או לגרסת Node.js.

הפקודה claude אינה מזוהה

סגרו את כל חלונות CMD ופתחו CMD חדש. הריצו claude doctor. במסלול npm ודאו שגם npm --version עובד.

npm מחזיר שגיאת הרשאות

פתחו CMD כמנהל מערכת ונסו שוב, או השתמשו במתקין המקורי המומלץ.

Node מותקן אבל npm אינו מזוהה

פתחו טרמינל חדש. אם עדיין לא עובד, התקינו מחדש LTS וסמנו הוספה ל-PATH.

איך יוצאים?

הקלידו /exit או לחצו Ctrl+C.

האם צריך מפתח API?

לא בהכרח. בהרצה הראשונה ניתן להתחבר דרך חשבון Anthropic מתאים. אופן החיוב תלוי בחשבון שבחרתם.
