DATABASE SCHEMA
================================================================================

Migration: migrations/003_free_diving_sessions.sql

Tables Created:
  1. free_diving_sessions (Main session data)
     - Core fields: title, description, location, date/time
     - Session details: type, level, capacity, price
     - Status tracking: scheduled, in_progress, completed, cancelled
     - Metadata: image_url, max_depth, instructor_id

  2. free_diving_session_bookings (Booking & payment)
     - Links: user → session
     - Payment: status, method, transaction_id, price
     - Booking: status, booked_at, attended

  3. free_diving_session_reviews (Ratings & reviews)
     - Rating: 1-5 stars with sub-ratings
     - Content: title, comment (min 10 chars)
     - Moderation: pending/approved/rejected
     - Metadata: helpful_count, reported flag

  4. free_diving_session_roster (Instructor attendance)
     - Attendance: attended (yes/no/null)
     - Tracking: check_in/check_out timestamps
     - Notes: instructor_notes field

Row-Level Security (RLS):
  ✓ Sessions: Public can view non-cancelled
  ✓ Instructors: Can update/delete own sessions only
  ✓ Bookings: Users see only their own bookings
  ✓ Reviews: Moderation-dependent visibility
  ✓ Roster: Instructors only see their rosters

Indexes Created:
  ✓ session_type, level, location, start_date
  ✓ user_id, status, payment_status, moderation_status
  ✓ instructor_id, is_verified, average_rating

API ENDPOINTS
================================================================================

Sessions Management:
  GET    /api/free-diving-sessions                List sessions (with filters)
  POST   /api/free-diving-sessions                Create session (instructor)
  GET    /api/free-diving-sessions/[id]           Get session details
  PATCH  /api/free-diving-sessions/[id]           Update session (instructor)
  DELETE /api/free-diving-sessions/[id]           Cancel session (instructor)

Booking System:
  POST   /api/free-diving-sessions/[id]/book      Book a session
  GET    /api/my-bookings                         Get user's bookings

Reviews:
  GET    /api/free-diving-sessions/[id]/reviews   Get session reviews
  POST   /api/free-diving-sessions/[id]/reviews   Submit review

Instructor:
  GET    /api/instructor/sessions                 Get instructor's sessions + roster

Filter Parameters (GET sessions):
  - sessionType: group_apnea_training, certification_course, etc.
  - level: beginner, intermediate, advanced, expert
  - location: text search
  - maxPrice: numeric limit
  - page: pagination
  - pageSize: results per page
