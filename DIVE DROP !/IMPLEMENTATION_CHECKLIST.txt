INTEGRATION INSTRUCTIONS
================================================================================

1. DATABASE
   Command: supabase db push
   Creates: Tables, functions, triggers, views, RLS policies

2. ENVIRONMENT
   Set: NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY, etc.
   And: CRON_SECRET for cron authentication

3. COMPONENTS
   Import and use in photo detail pages
   Example: <RatingForm photoId={id} onSubmit={handleRate} />

4. HOOKS
   Use usePhotoRating(photoId) for complete functionality
   Or usePhotoStats() and usePhotoRatings() for specific data

5. CRON JOB
   Configure daily execution at 2 AM UTC
   Recalculates scores and updates stats

6. TESTING
   Submit test ratings
   Verify stats update
   Check admin dashboard
